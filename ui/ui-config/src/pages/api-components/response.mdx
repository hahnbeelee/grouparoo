## ResponseField Component

```jsx
<ResponseField name="username" type="string" required>
  The username of the user
</ResponseField>
```

### Properties

| Name       | Type    | Description                                      |
| ---------- | ------- | ------------------------------------------------ |
| `name`     | string  | The name of the response value                   |
| `type`     | string  | Expected type of the response value              |
| `required` | boolean | Indicate whether the response is always expected |
| `children` | string  | Description of the response (markdown enabled)   |
