## Request Example

The `<RequestExample>` component works similar to [CodeGroup](/components/code#code-group), but displays the request content on the right sidebar

````md
<RequestExample>

```bash Request
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````

## Response Example

The `<ResponseExample>` component works similar to [CodeGroup](/components/code#code-group), but displays the response content on the right sidebar

````md
<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>
````

<RequestExample>

```bash Request
curl --request POST \ --url https://dog-api.kinduff.com/api/facts \
```

</RequestExample>

<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>

<Note>

Make sure to add add an extra line at the start and end of the code block if it's not working properly.

</Note>
