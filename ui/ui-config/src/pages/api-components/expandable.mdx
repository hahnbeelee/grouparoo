## Expandable component

The `<Expandable>` component lets you hide nested properties for the user to toggle

```jsx
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      The full name of the user
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Whether the user is over 21 years old
    </ResponseField>

    <ResponseField name="age" type="number">
      The last name of the user
    </ResponseField>
  </Expandable>
</ResponseField>
```

### Properties

| Name   | Type   | Description                               |
| ------ | ------ | ----------------------------------------- |
| `name` | string | The name of the set of elements to toggle |
