---
title: "Amplitude Integration"
description: "Mintlify can send analytics to Amplitude."
api: "PUT https://docs.mintlify.com/api/v1/sites/integrations/amplitude"
auth: "bearer"
---

<RequestExample>

```json Request
curl -X PUT https://docs.mintlify.com/api/v1/sites/integrations/amplitude
    -H "Authorization: Bearer YOUR_MINTLIFY_ACCESS_TOKEN"
    -H "Content-Type: application/json"
    -d '{"apiKey":"YOUR_AMPLITUDE_API_KEY"}'
```

</RequestExample>

<ResponseExample>

```md Response
Set Amplitude apiKey.
```

</ResponseExample>

Set your API key by making a `PUT` request to our endpoint. We will start sending analytics on your next site deployment.

<ParamField body="apiKey" type="string" required>
  Your Amplitude API key.
</ParamField>

You can make a `GET` request to verify you set the API key correctly. You can also use a `DELETE` request to remove the integration.
