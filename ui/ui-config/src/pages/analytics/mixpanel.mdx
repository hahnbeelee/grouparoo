---
title: "Mixpanel Integration"
description: "Mintlify can send analytics to Mixpanel."
api: "PUT https://docs.mintlify.com/api/v1/sites/integrations/mixpanel"
auth: "bearer"
---

<RequestExample>

```json Request
curl -X PUT https://docs.mintlify.com/api/v1/sites/integrations/mixpanel
    -H "Authorization: Bearer YOUR_MINTLIFY_ACCESS_TOKEN"
    -H "Content-Type: application/json"
    -d '{"projectToken":"YOUR_MIXPANEL_PROJECT_TOKEN"}'
```

</RequestExample>

<ResponseExample>

```md Response
Set Mixpanel projectToken.
```

</ResponseExample>

Set your project token by making a `PUT` request to our endpoint. We will start sending analytics on your next site deployment.

<ParamField body="projectToken" type="string" required>
  Your Mixpanel project token.
</ParamField>

You can make a `GET` request to verify you set the project token correctly. You can also use a `DELETE` request to remove the integration.
