## Initialize Request

### Parameters

<ParamField body="code" type="string" required>
  The code that a docstring should be generated for
</ParamField>

<ParamField body="languageId" type="string" required>

The [VSCode language identifier](https://code.visualstudio.com/docs/languages/identifiers#_known-language-identifiers)

</ParamField>

<ParamField body="fileName" type="string" required>
  The name of the file. Note: the extension is used to identify the language
  when languageId is invalid
</ParamField>

<ParamField body="commented" type="boolean" default="false">
  Whether comments should be generated for the results in the language requested
</ParamField>

<ParamField body="userId" type="string" required>
  A unique user identification string. Can be any non-empty value
</ParamField>

<ParamField body="email" type="string" required>
  The email you used to request API access. Note: this is used to bypass regular
  usage quotas
</ParamField>

<ParamField body="context" type="string">

The file or snippet containing `code` that can provide more context for the generation

</ParamField>

<ParamField body="width" type="number" default={100}>

The maximum number of characters for each line of generated results. Used so that results aren't all in the same line.

</ParamField>

<RequestExample>

```javascript JavaScript
const {
  data: { id },
} = await axios.post("https://api.mintlify.com/docs/write/v3", {
  code: 'console.log("Hello world")',
  languageId: "javascript",
  fileName: "index.js",
  commented: true,
  userId: "123",
  email: "hi@mintlify.com",
  context: 'function print() {\nconsole.log("Hello world")\n}',
  width: 100,
});
```

</RequestExample>

### Response

<ResponseField name="id" type="number">

The id of the job initiated to track the docstring generation progress and fetch results.

</ResponseField>

<ResponseExample>

```json Example Response
{
  "id": 494282
}
```

</ResponseExample>
